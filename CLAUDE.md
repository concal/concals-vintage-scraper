# Scrapinator

A personal vintage clothing scraper and storefront. Scrapy spiders scrape product listings from vendor sites, store them in MongoDB, and a FastAPI backend serves them to a React frontend.

## Repo structure

```
backend/   FastAPI app + Scrapy scraper
frontend/  React + Vite + Tailwind + shadcn/ui
```

## Frontend

I am not a web designer, so I am not an expert on design decisions. Use your own judgement to leverage shadcn and tailwind tools to make the design of this web app look good. Users for this website are into archival designer fashion, high end vintage, and secondhand clothing. The userbase is "cool", so the design of this webapp should reflect that and appeal to their tastes.

**Stack:** React 19, TypeScript, Vite, Tailwind v4 (`@tailwindcss/vite` plugin), shadcn/ui (style: `radix-nova`, preset: `b1hbqS`)

**Path alias:** `@/` → `src/`

**shadcn convention:** All code generated by `npx shadcn` lives in `src/generated/`. Hand-written components live in `src/components/`. Never put hand-written code in `src/generated/`, and never put generated code outside it.

To add a new shadcn component:

```
cd frontend && npx shadcn@latest add <component>
```

It will land in `src/generated/components/ui/` automatically (configured in `components.json`).

**Dev server:** `cd frontend && npm run dev`
**Build:** `cd frontend && npm run build`

## Backend

**Stack:** FastAPI, Scrapy, MongoDB (via pymongo), python-dotenv

**Entry point:** `backend/api/main.py`
**Routes:** `backend/api/routes.py` — all under `/products` prefix
**Scraper:** `backend/api/product_scraper/`

**Run the API:**

```
cd backend && uvicorn api.main:app --reload
```

**Deployed to:** Koyeb (`https://reluctant-lura-concal-e5f49e86.koyeb.app`)

**Environment variables** (`.env` at repo root):

- `MONGO_URI`
- `MONGO_DB_NAME`
- `MONGO_COLLECTION_NAME`

## API endpoints

| Method | Path                       | Description                       |
| ------ | -------------------------- | --------------------------------- |
| POST   | `/products/search`         | Filtered + paginated product list |
| POST   | `/products/save`           | Save a product by index           |
| POST   | `/products/unsave`         | Unsave a product by index         |
| GET    | `/products/saved-products` | Get saved product indexes         |
| GET    | `/products/scrape`         | Trigger scraper (background task) |

The frontend hits the deployed Koyeb URL in production. The local URL (`http://127.0.0.1:8000`) is commented out in `src/api/products.ts`.
